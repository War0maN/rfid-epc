# Хэрэглэгчийн гарын авлага

RFID EPC системийн өдөр тутмын хэрэглээний заавар — таб бүрээр. (Таны эрхээс хамаарч зарим таб/товч харагдахгүй байж болно — админаас эрх авна.)

## Нэвтрэх

Админы урьсан **имэйлээрээ** "Бүртгүүлэх" хийхэд автоматаар байгууллагад нэгдэнэ. Дараа нь имэйл + нууц үгээрээ нэвтэрнэ.

---

## Шинэ ажил — Excel импорт

Ирсэн барааны packing list-ийг оруулж EPC бөөнөөр үүсгэнэ.

1. Ажлын дугаар, ирсэн огноо, нийлүүлэгч, салбараа бөглөнө
2. Excel файлаа сонгоно. Багана (толгойн нэр уян хатан танигдана):
   - `name` (нэр) · `sku` · `barcode` (баркод) · `price` (үнэ) · **`piece` (тоо ширхэг — заавал)** · `box` (хайрцаг) · `branch` (салбарын код/нэр — мөр бүрд өөр байж болно) · `category` (ангилал: "Хувцас / Дээд хувцас")
   - Бусад бүх багана = шинж чанар (Өнгө, Размер...) болж автоматаар бүртгэгдэнэ
3. Импортлоход бараа/ангилал автоматаар бүртгэгдэж, ширхэг бүрд EPC үүснэ (төлөв: **Хэвлээгүй**)

## Бүтээгдэхүүн

- **Бүтээгдэхүүн** дэд таб: барааны жагсаалт (үлдэгдэл = идэвхтэй EPC тоо). "+ Бараа нэмэх"-ээр гараар бүртгэх, "EPC үүсгэх"-ээр нэмэлт ширхэг үүсгэх (салбар + тоо)
- **Ангилал** дэд таб: 3 түвшний ангилал ба глобал шинж чанарын удирдлага

## Үлдэгдэл

Идэвхтэй (хэвлэгдсэн, зараагүй) барааны тоо — **бараа × салбар** матрицаар, Нийт тоо + Нийт үнийн дүнтэй.

- Салбарын **тоон дээр дарвал** тухайн EPC-үүдийн жагсаалт (CSV татаж болно)
- ⚙ Багана-аар салбар нуувал Нийт/дүн зөвхөн харагдаж буйгаар тооцогдоно
- "Бүгд харах" — үлдэгдэлгүй барааг ч харуулна

## Гүйлгээ

**Гүйлгээ** дэд таб — шинэ гүйлгээ (скан → сагс):

1. **Төрөл** сонгоно: Борлуулсан / Шилжүүлэг / Буцаалт / Бусад гүйлгээ
2. **Салбар (эх)** сонгоно → доор тохирох EPC-ийн жагсаалт гарна (Буцаалтад: Борлуулсан/Бусад гүйлгээт бараа)
3. EPC-г **RFID уншигчаар уншуулах** / гараар шивэх (Enter) / жагсаалтаас "+ Нэмэх" дарж сагсанд хийнэ
4. Сагсны нийт тоо·дүнг хараад **Баталгаажуулна**

- **Шилжүүлэг**: очих салбараа сонгоно; бараа "Шилжүүлж буй" болно. Хүлээн авах салбар **Гүйлгээний түүх**-ээс "Хүлээн авах" дарж авна (эсвэл "Цуцлах" — эх салбартаа буцна)
- **Буцаалт**: зарагдсан/гарсан бараа буцаж ирэхэд — Идэвхтэй болж үлдэгдэлд орно
- **Гүйлгээний түүх** дэд таб: бүх гүйлгээ; мөр дарвал ширхэгүүд нь (CSV татна). Гүйлгээ устгагддаггүй — алдааг буцаалт/цуцлалтаар залруулна

## Тайлан

Борлуулалтын тайлан: огнооны интервал (Өнөөдөр / 7 хоног / 30 хоног / Энэ сар) + **Бүлэглэх** (Өдрөөр / Сараар / Салбараар / Бараагаар / Хэрэглэгчээр). Нийт дүн/ширхэг/гүйлгээний тоо, график, хүснэгт, CSV. Үнэ нь **борлуулсан үеийн үнээр**.

## Бараа (EPC)

- **Жагсаалт** дэд таб: бүх EPC — багана бүрээр шүүх, эрэмбэлэх, ⚙ багана нуух
  - **CSV татах / ZPL татах / 🖨 Хэвлэх** — сонгосон эсвэл шүүлтэд тохирох бүгдэд. Хэвлэхэд бараа автоматаар **Идэвхтэй** болно
  - **Төлөв өөрчлөх** (админ): сонгосон/шүүсэн EPC-ийн төлөвийг тэмдэглэлтэй солино
  - **Устгах** (админ): checkbox-оор сонгоод — зөвхөн Хэвлээгүй/Идэвхтэй устгагдана; зарагдсан болон гүйлгээний түүхтэй нь хамгаалагдана
  - **EPC код дээр дарвал** → Хайлт руу үсэрч бүрэн түүхийг харуулна
- **Хайлт** дэд таб: RFID-аас уншсан 24 тэмдэгт hex-ээр хайж, барааны мэдээлэл + **бүрэн түүх** (үүссэн → хэвлэсэн → шилжүүлсэн → зарагдсан → буцаагдсан... хэн/хэзээ/тэмдэглэлтэй) харна

## Шошго

Шошгоны загвар зохиох (дизайнер) ба Zebra принтерээр хэвлэх тохиргоо.

## Салбар

Агуулах/дэлгүүрийн салбарууд: нэр + код (код нь импортын `branch` баганад ашиглагдана). EPC-тэй салбар устгагдахгүй.

## Аудит

Бүх үйлдлийн түүх: хэн/хэзээ/юу. **Мөр дээр дарвал** дэлгэрэнгүй: яг аль бараа/салбар/ажил, бөөн үйлдэлд бараагаар задаргаа + EPC жагсаалт, засварт Өмнө/Дараа утга.

## Хэрэглэгчид (зөвхөн админ)

- **Урих**: имэйл + эрх (Админ/Оператор). Урилга имэйл илгээгддэггүй — уригдсан хүнд "энэ имэйлээрээ бүртгүүлээрэй" гэж хэлнэ
- **Салбар** товч: операторыг салбар(ууд)-д хуваарилна — зөвхөн тэр салбарын өгөгдлийг харж, гүйлгээ хийнэ. Хоосон = бүх салбар
- **Эрх** товч: таб харах + үйлдлийн эрхийг checkbox-оор. Бүгд чеклэгдсэн = бүрэн эрх (default)

## Түгээмэл асуулт

**Буруу зарчихлаа — яах вэ?** Гүйлгээ → Буцаалт хийнэ (эсвэл админ Төлөв өөрчлөх). Түүхэнд хоёулаа бүртгэгдэнэ.

**Импортын дараа Үлдэгдэл 0 байна?** Үлдэгдэл = зөвхөн **хэвлэгдсэн (Идэвхтэй)** бараа. Шошгоо хэвлэсний дараа орж ирнэ.

**Устгаж болохгүй гэж байна?** Зарагдсан/гүйлгээтэй зүйлс түүхэн дата тул хамгаалагдсан — энэ нь зориудын хамгаалалт. Цэвэрлэх бол: Ажлыг устгах (хэвлээгүй EPC нь хамт устана) эсвэл Хэвлээгүй/Идэвхтэй EPC-г сонгож устгах.
